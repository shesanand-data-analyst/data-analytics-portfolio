# SQL Projects

## Sales Performance Analysis
This project analyzes sales data using SQL to answer key business questions related to product demand, revenue trends, growth, and branch performance.

### Key Focus Areas
- Data validation and preparation
- Feature engineering using views
- Product performance and demand analysis
- Revenue trends and Month-over-Month growth
- Branch benchmarking and segmentation
- Advanced SQL techniques (CTEs, subqueries, window functions)

> Detailed queries and insights are documented directly inside the `sales-analysis.sql` file.
